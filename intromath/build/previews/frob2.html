Finally, every definition establishes a bi-implication between the newly-defined terminology and its meaning. By convention, definitions usually leave the “only if” implicit. For example, we write “a squirrel is a frob if bla” rather than “a squirrel is a frob if and only if bla”. This is technically incorrect, but so widely spread that there is little point in fighting it16
