Definition 3.1: Set

A set is a collection of objects that ignores the order of those objects and how often they occur.

We denote a set by listing its objects within curly braces, separated by commas, for example: {1,2,3}\htmlData{preview=http://localhost:1234/previews/set.html}{\href{http://localhost:1234/sets.html#set}{\lbrace}}1, 2, 3\htmlData{preview=http://localhost:1234/previews/set.html}{\href{http://localhost:1234/sets.html#set}{\rbrace}}{1,2,3}.

The empty set {}\htmlData{preview=http://localhost:1234/previews/set.html}{\href{http://localhost:1234/sets.html#set}{\lbrace}}\htmlData{preview=http://localhost:1234/previews/set.html}{\href{http://localhost:1234/sets.html#set}{\rbrace}}{} — the collection that contains no objects at all — is also written as ∅\htmlData{preview=http://localhost:1234/previews/set.html}{\href{http://localhost:1234/sets.html#set}{\emptyset}}∅.

Because a set ignores the order of its objects, {1,2}\htmlData{preview=http://localhost:1234/previews/set.html}{\href{http://localhost:1234/sets.html#set}{\lbrace}}1, 2\htmlData{preview=http://localhost:1234/previews/set.html}{\href{http://localhost:1234/sets.html#set}{\rbrace}}{1,2}, and {2,1}\htmlData{preview=http://localhost:1234/previews/set.html}{\href{http://localhost:1234/sets.html#set}{\lbrace}}2, 1\htmlData{preview=http://localhost:1234/previews/set.html}{\href{http://localhost:1234/sets.html#set}{\rbrace}}{2,1} both denote the same set. Since a set also ignores how often its object occur, {1,2}\htmlData{preview=http://localhost:1234/previews/set.html}{\href{http://localhost:1234/sets.html#set}{\lbrace}}1, 2\htmlData{preview=http://localhost:1234/previews/set.html}{\href{http://localhost:1234/sets.html#set}{\rbrace}}{1,2} and {1,2,1}\htmlData{preview=http://localhost:1234/previews/set.html}{\href{http://localhost:1234/sets.html#set}{\lbrace}}1, 2, 1\htmlData{preview=http://localhost:1234/previews/set.html}{\href{http://localhost:1234/sets.html#set}{\rbrace}}{1,2,1} also denote the same set. All three expressions are simply different ways of denoting the unique mathematical object characterized by the fact that 111 and 222 belong to it, but nothing else does.

We now introduce more succinct terminology for talking about which objects do and which objects do not belong to a particular set.

We can visualize set membership by drawing sets as blobs that enclose their elements. Example 3.4 demonstrates such an Euler diagram. Note how the idea of a set containing elements multiple times or in a particular order cannot even be expressed in these depictions, making Euler diagrams a more faithful representations of sets than our written notation.
